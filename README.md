# Object-Sorting-Robotic-Arm
- A robotic arm that picks and places blocks according to its size and color. 
- The hardware was controlled by Arduino. It uses 4 metal gear MG995 servo motors for the Base, Waist, Arm and the Gripper base and plastic gear SG-90 servo motor for the gripper. 
- Originally 6 Degree of Freedom (DoF) according to the STL files, later modified to 5 DoF, where the gripper and gripper base were attached for adjustment. The Arduino code is according for 5 DoF written to fit the adjustment. 
- The STL files were freely available online. Refer to "How to Mechatronics" Youtube channel.
